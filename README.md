## Exercise 1

**Problem:**

Count negative numbers in a matrix 

**Example:**

	Input:
	Matrix: [[-3, 5, -2], 
		 [7, -8, 6], 
		 [-4, 1, -3]]
	Output: 5
	
	Input:
	Matrix: [[10, 8, -5], 
		 [-6, 4, 2], 
		 [1, -1, 0]]
	Output: 3
	
	Input:
	Matrix: [[0, 0, 0], 
		 [0, 0, 0], 
		 [0, 0, 0]]
	Output: 0

## Exercise 2

**Problem:**

Find the row with the maximum sum 

**Example**

	Input:
	Matrix: [[3, 5, 2], 
		 [7, 8, 6], 
		 [4, 1, 3]]
	Output: 1
	
	Input:
	Matrix: [[10, 8], 
		 [6, 4]]
	Output: 0
	
	Input:
	Matrix: [[1, 2, 3], 
		 [4, 5, 6], 
		 [7, 8, 9]]
	Output: 2

## Exercise 3

**Problem:**

Find the largest difference between any two elements

**Example**

	Input:
	Matrix: [[3, 5, 2], 
		 [7, 8, 6], 
		 [4, 1, 3]]
	Output: 7
	
	Input:
	Matrix: [[10, 8], 
		 [6, 4]]
	Output: 6
	
	Input:
	Matrix: [[-10, 20, 15], 
		 [7, -8, 6], 
		 [-4, 1, -3]]
	Output: 30
